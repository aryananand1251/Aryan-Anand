Featured Projects

IPv4V6 (Android App)

This application is designed to simplify and automate the process of calculating network addresses, subnet masks, broadcast addresses, and the number of available hosts based on a given IP address and subnet. It serves as a valuable tool for networking students, IT professionals, and system administrators.

Uber review service (Backend)

This project is a backend application designed to manage and process user reviews for Uber-like ride-sharing services. Built using Spring Boot, it follows RESTful architecture and integrates with Hibernate and JPA for efficient ORM-based database operations. Users can submit reviews and ratings for drivers and services, which are securely stored and managed in a SQL database.

Resume Wala

Resume Wala is an AI-powered resume-building application designed to help students and professionals create ATS-friendly (Applicant Tracking System) resumes tailored to specific job descriptions. By simply uploading an existing resume or entering job-related details, users can generate optimized resumes that improve their chances of passing through automated hiring systems.

Keep Notes (Android Application)

This project is a feature-rich Notes application developed using Java and XML for Android. The app allows users to efficiently manage their notes with full CRUD operations—Create, Read, Update, and Delete—using SQLite integrated via Room Database for robust data persistence.
